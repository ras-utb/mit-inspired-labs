Introduction & Safety
(1 minute read)

3D printing is an additive manufacturing process where computer-aided G-code is translated to build up material layer by layer. It includes FDM (extruding hot thermoplastics), SLA (photoresin polymers cured with light), and Polyjet printing.

In this lab, you will learn how to:

  • ✅ Learn advanced 3D modeling techniques (Revolve, Sweep, Warp)
  • ✅ Design for multimaterial 3D printing
  • ✅ CAD a rotating element around a shaft
  • ✅ CAD a threaded object
  • ✅ Make a mechanical assembly
  • ✅ Render a 3D model in Fusion360
  • ✅ Export files for 3D printing

Part (2): Sketch and Dimension Parametrically (5 minutes)

Instructions:

  • Draw a circle at the origin (0,0)
  • Dimension the circle using “shaft_outer_d”
  • Exit sketch and rename it shaft-circle

Part (3): Extruding the Sketch (5 minutes)

Steps:

  • Select Create → Extrude
  • Select the circle from your sketch
  • Set distance to shaft_length_3b or shaft_length_4b
  • Operation: New Component
  • Rename component to Shaft

Part (4): Making a Threaded Hole (7 minutes)

Video: How to Make a Threaded Hole

Steps:

  • Toggle “shaft-circle” sketch ON
  • Create → Hole
  • Select the top face
  • Reference: center point of circle
  • Hole Type: Simple
  • Tap Type: Tapped
  • Thread Offset: Full
  • Drill Point: Angle
  • Depth: thread_depth
  • Size: 7mm
  • Designation: M7mm x 1mm
✔ Checkoff 2 — Show your shaft with threaded hole

Part (5): Creating Profile of Keychain Holder Head (20 minutes)

Video: Click here to watch

Steps:

  • Create → Axis Through Cylinder
  • Create → Plane at Angle → set angle to 0°
  • Right-click plane → Create Sketch
  • Project the circle boundary (keep Projection Link ON)
  • Draw your remix profile (must be closed + symmetric)
