Introduction & Safety (1 min read)

3D printing is an additive manufacturing process where G-code instructs the machine to build material layer by layer. Techniques include FDM (hot thermoplastics), SLA (photoresin), and Polyjet 3D printing.

In this lab, you will learn to:

  • Learn advanced 3D modeling Techniques (Revolve, Sweep, Warp)
  • Design for Multimaterial 3D Printing
  • CAD a simple rotating element around a shaft
  • CAD a threaded object
  • Make a Mechanical Assembly
  • Render a 3D Model in Fusion360
  • Export files and prepare them for 3D printing

Fabrication Quest of the Day

Today, you will:

  • Assemble your Rotating Keychain
  • Post-process your 3D print

Machines: 3D Printer

Part (1): Creating Variables & Equations for Dynamic Dimensioning (10 min)

Dimensioning models through variables allows easy modification. Go to Modify → Change Parameters → Add User Parameter (+ sign). Create variables as below:

Checkoff 1: Ask your instructor to check your Part (1) setup.

Part (2): Sketch and Dimension Parametrically (Sketching Keychain Shaft, 5 min)

  • Draw a circle at the center (0,0).
  • Dimension the circle with “shaft_outer_d”.
  • Exit Sketch, rename to "shaft-circle".

Part (3): Extruding the Sketch (5 min)

  • Create → Extrude Tool → Select circle from "shaft-circle".
  • Direction: One Side
  • Distance: shaft_length_3b or shaft_length_4b
  • Operation: New Component
  • Rename component → "Shaft"

Part (5): Creating Profile of Keychain Holder Head (20 min, Remix Optional)

  • Create Axis Through Cylinder (Construct → Axis Through Cylinder).
  • Construct → Plane at Angle → Angle = 0
  • Create New Sketch → Project → Project cylinder top perimeter.
  • Draw profile for top holder (closed & symmetric)
  • Finish Sketch → Revolve to create top keychain holder
Checkoff 3: Show your sketched profile to instructor.

Part (7): Making Hole for Keychain (5 min)

  • Create Tangent Plane
  • Sketch Circle → Dimension = 4mm
  • Create → Extrude → Cut → Symmetric Direction
Checkoff 4: Show keychain hole to instructor.

Part (8): Understanding Sketch Relationships & Constraints (5 min)

Learn how sketches relate to each other. Constraints ensure your parts behave predictably when dimensions change.

Part (9) [OPTIONAL]: Wrapping/Embossing a Design on Cylindrical Shaft (5 min)

  • Profile: Select sketch to emboss.
  • Face: Select surface to emboss onto.
  • Emboss Type: Emboss/Add or Deboss/Cut
  • Depth: Set desired height/depth
