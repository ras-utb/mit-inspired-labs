Introduction & Safety

3D printing is an additive manufacturing process where computer-aided G-code is translated to build material layer by layer. Techniques include FDM (hot thermoplastics), SLA (photoresin polymers), and Polyjet 3D printing.

In this lab, you will learn to:

  • ✅ Learn advanced 3D modeling Techniques (Revolve, Sweep, Warp)
  • ✅ Design for Multimaterial 3D Printing
  • ✅ CAD a simple rotating element around a shaft
  • ✅ CAD a threaded object
  • ✅ Make a Mechanical Assembly
  • ✅ Render a 3D Model in Fusion360
  • ✅ Export files and prepare them for 3D printing

Fabrication Quest of the Day

  • (a) Assemble your Rotating Keychain
  • (b) Post-process your 3D print

Software & Hardware

Software: Fusion360

Machines: 3D Printer

Part 1: Creating Variables & Equations for Dynamic Dimensioning (10 min)

Video Summary

Dimensioning models through variables allows for model dimension modification with ease.

To create dimension variables go to: Modify → Change Parameters → Add User Parameter (“+” sign).

Create a variable for “shaft_outer_d” representing the inner diameter of the Threaded Shaft (9mm). Add the following variables:

Shaft Outer Diametershaft_outer_d9mm

Part 2: Sketch and Dimension Parametrically (5 min)

  • Draw a circle at the center (0,0).
  • Dimension the first circle with “shaft_outer_d”.
  • Exit Sketch and rename it to "shaft-circle".

Part 3: Extruding the Sketch (5 min)

  • Click on Create.
  • Select the Extrude Tool.
  • Under Profile, select the circle from "shaft-circle".
  • Set Direction to One Side.
  • Type Distance as “shaft_length_3b” or “shaft_length_4b”.
  • Choose operation as New Component.
  • Rename component to "Shaft".

Part 4: Making a Threaded Hole (7 min)

Video Summary
  • Toggle "Shaft-circle" sketch on.
  • Go to Create → Hole Tool.
  • Select top face of cylinder.
  • Set Hole Type to Simple, Tap Type to Tapped, Depth to "thread_depth".
  • Size: 7mm, Designation: M7mmx1mm, Click Ok.

Checkoff 2: Show your shaft with threaded hole to your instructor.

Part 5: Creating Profile of Keychain Holder Head (20 min)

Video Summary
  • Create Axis Through Cylinder: Construct → Axis Through Cylinder → select cylinder.
  • Create Plane at Angle: Construct → Plane at Angle → select axis → angle 0.
  • Create New Sketch on the Plane, project top circle perimeter.
  • Draw symmetric profile for top holder.
  • Finish Sketch → Revolve to create top keychain holder.

Checkoff 3: Show & explain your sketched profile to instructor.
