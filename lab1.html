Introduction & Safety

3D printing is an additive (add material) manufacturing process where computer-aided G-code is translated to build material layer by layer. Techniques include FDM, SLA, and Polyjet printing.

In this lab, you will:

  • ✅ Learn advanced 3D modeling Techniques (Revolve, Sweep, Warp)
  • ✅ Design for Multimaterial 3D Printing
  • ✅ CAD a simple rotating element around a shaft
  • ✅ CAD a threaded object
  • ✅ Make a Mechanical Assembly
  • ✅ Render a 3D Model in Fusion360
  • ✅ Export files for 3D printing

Part (2): Sketch and Dimension Parametrically (5 minutes)

  • Draw a circle at the center (0,0)
  • Dimension the circle with “shaft_outer_d”
  • Exit Sketch and rename it "shaft-circle"

Part (3): Extruding the Sketch (5 minutes)

  • Click on Create → Extrude Tool
  • Select circle profile
  • Set Direction to One Side
  • Type distance as “shaft_length_3b” or “shaft_length_4b”
  • Operation: New Component
  • Rename component to "Shaft"

Part (4): Making a Threaded Hole (7 minutes)

Video Link

  • Toggle "Shaft-circle" sketch back on
  • Create → Hole Tool, select top face
  • Reference: center of circle
  • Hole Type: Simple, Tap Type: Tapped
  • Thread Offset: Full, Drill Point: Angle, Depth: "thread_depth"
  • Size: 7mm, Designation: M7mmx1mm, Modeled: ON
  • Click OK

Checkoff 2: Show your shaft with its threaded hole to your instructor.

Part (5): Creating Profile of Keychain Holder Head (20 minutes)

Video Link

  • Axis Through Cylinder → Construct → Axis
  • Plane at Angle → Construct → Plane (angle 0)
  • Sketch profile on plane, project geometry from top circle
  • Finish Sketch → Revolve to create top holder

Checkoff 3: Show your sketched profile to your instructor.

Part (6): Revolving Radially Symmetric Geometry (5 minutes)

Video Link

  • Create → Revolve
  • Select profile
  • Select axis of symmetry (cylinder axis)
  • Set angle 360°
  • Click OK

Part (7): Making Hole for Keychain (5 minutes)

Video Link

  • Create Tangent Plane → Construct
  • Sketch circle for hole, diameter 4mm
  • Extrude Cut → Symmetric

Checkoff 4: Show your shaft's keychain hole to your instructor.
