Lab 3: Let's Make a Guitar Hero Game!

Part (1): Getting a Preliminary GUI

Processing uses a programming language called Java. Just like Arduino, it has two main functions: setup() and draw(). setup() runs once at the beginning. draw() runs continuously at 60 frames per second, letting you create animations. To make a basic window, we will use the ControlP5 library, which contains pre-built functions for GUIs.

Library documentation: https://processing.org/download

Basic window code in Processing:

import controlP5.*;

ControlP5 cp5;

void setup(){
    size(300, 400);
}

void draw(){
    background(50, 70, 100);
    fill(0, 0, 0);
    text("GUITAR HERO", 100, 30);
}

You should see a dark blue window with "GUITAR HERO".

Now, let's add a button and font:

import controlP5.*;
PFont font;

ControlP5 cp5;

void setup(){
    size(300, 400);
    font = createFont("ComicSansMS", 20);
    cp5 = new ControlP5(this);
    cp5.addButton("START")
        .setPosition(100, 50)
        .setSize(100, 80)
        .setFont(font);
}

void draw(){
    background(50, 70, 100);
    fill(0, 0, 0);
    text("GUITAR HERO", 100, 30);
}

Rerun and you should see a "START" button.

Finally, add a simple circle animation:

import controlP5.*;
PFont font;

ControlP5 cp5;
float circleY = 0;

void setup(){
    size(300, 400);
    font = createFont("ComicSansMS", 20);
    cp5 = new ControlP5(this);
    cp5.addButton("START")
        .setPosition(100, 50)
        .setSize(100, 80)
        .setFont(font);
}

void draw(){
    background(50, 70, 100);
    ellipse(100, circleY, 50, 50);
    circleY = circleY + 1;
    if(circleY > height) {
      circleY = 0;
    }
    fill(0, 0, 0);
    text("GUITAR HERO", 100, 30);
}

You should see a moving circle on the window. Awesome, now you know how to make a basic window, add a button, and create a simple animation.
