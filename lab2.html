Lab Overview

This is the first in a two‑part robotics lab. Today you will assemble and program your own robot!

Part 1: Understanding Code & Testing Motors

  1. Install the Adafruit Motor Shield library in Arduino IDE (Books icon → Library Manager → search and install).
  2. Add this at the top of your Arduino sketch:
    #include <AFMotor.h>
  3. Create motor objects above setup(): 
    AF_DCMotor rightMotor(1);
AF_DCMotor leftMotor(2);
  4. Set motor speeds in setup() using setSpeed() (0–255).
  5. Set directions using run(FORWARD), BACKWARD, or RELEASE.
  6. Plug the shield onto the Arduino (align pins!), connect the yellow TT motors to M1 and M2.
  7. Upload your code — motors should spin slowly (USB power is weak).
  8. Now modify your code to:
    • Drive forward 3 seconds
    • Stop 1 second
    • Drive backward 3 seconds
    • Repeat (inside loop())

Checkoff 1

Ask your instructor for a checkoff on completing Part (1). Demonstrate your forward → stop → backward script.

Part 2: Assembling the Chassis

Confirm you have all the parts, then follow the build steps carefully.

  1. Attach the ball caster using the pointed screws.
  2. Mount the Arduino (without shield), then add the shield.
  3. Mount the servo at the front.
  4. Attach motor brackets and insert nuts.
  5. Attach motors to brackets.
  6. Attach battery pack to bottom with double‑sided tape (switch facing down).
  7. Add mini breadboard & mount ultrasonic sensor.
  8. Add wheels (push fully but gently).
  9. Now connect electronics:
    • Feed servo wires through back hole → plug into SERVO port (yellow = signal).
    • Feed motor wires → M1 = left, M2 = right.
    • Attach battery wires to POWER terminals (do NOT swap).

Checkoff 2

Ask your instructor for a checkoff on completing Part (2). Batteries will be provided afterward.

Part 3: Driving Straight

  1. Try driving your robot straight using the same motor speed for both motors.
  2. Adjust relative speeds until it drives reasonably straight.
  3. Create a function:
    void driveStraight(int millis) {
    // fill in using your corrected speeds
}
  4. Test with 0.5s, 1s, and 3s drives.

  2. Fill in:

Part 5: Driving a Loop

Use your driving + turning functions to:

  • Drive a 50cm square clockwise
  • Drive the same square counterclockwise
