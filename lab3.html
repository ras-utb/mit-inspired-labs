Lab 3: Guitar Hero — Session 1

Building the Guitar Hero controller & integrating with Processing

Home Labs Instructors About

This lab teaches you how to build a Guitar Hero controller with buttons and sensors, connect it to an Arduino, and interface with Processing for gameplay. By the end, you will have a playable mini Guitar Hero setup and understand how hardware input controls software in real-time.

Safety & Important Tips (~1 minute read)

  1. Disconnect Arduino from USB or power supply when wiring buttons or sensors.
  2. Double-check polarity when connecting LEDs and buzzers.
  3. Do not short any wires or sensors to avoid damage.
  4. Use resistors where instructed to prevent burning components.

Components you will need (provided by instructors):

  • Arduino Uno + USB cable
  • Push buttons (x5 for frets)
  • Piezo buzzer (optional)
  • Wires & breadboard
  • LEDs (optional)
  • Enclosure box / guitar outline

Start by wiring the buttons to the Arduino:

  1. Connect one side of each push button to a digital pin (e.g., D2–D6).
  2. Connect the other side to GND.
  3. Use pull-up resistors in code or hardware to avoid floating pins.

Upload the basic test code to Arduino to verify each button press prints to Serial Monitor:

const int buttonPins[] = {2,3,4,5,6};
void setup() {
  Serial.begin(9600);
  for(int i=0;i<5;i++){ pinMode(buttonPins[i], INPUT_PULLUP); }
}
void loop() {
  for(int i=0;i<5;i++){
    if(digitalRead(buttonPins[i]) == LOW){
      Serial.println("Button " + String(i+1) + " pressed");
      delay(200);
    }
  }
}

Checkoff 1: Ask your instructor to verify button wiring and Serial Monitor output.

Next, integrate Arduino with Processing:

  1. Open Processing and install the Serial library if not present.
  2. Use the following code to read Arduino input and visualize notes:
import processing.serial.*;
Serial arduino;
void setup(){
  size(600,400);
  arduino = new Serial(this, "COM3", 9600);
}
void draw(){
  background(0);
  if(arduino.available() > 0){
    String inData = arduino.readStringUntil('\n');
    if(inData != null){
      println(inData);
      // visualize pressed buttons here
    }
  }
}
